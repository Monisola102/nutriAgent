#  NutriBot Agent

NutriBot is an intelligent nutrition assistant built with the Mastra framework.
It provides users with accurate nutritional information and health benefits of foods.
---

##  Overview

NutriBot helps users make smarter food choices by analyzing the nutritional value and health benefits of various foods.
